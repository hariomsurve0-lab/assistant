# 🤖 Jarvis — AI Personal Assistant

> A personal AI-powered assistant for Gopal's laptop, inspired by Iron Man's Jarvis.  
> Built to automate tasks, manage files, run scripts, and make digital life faster and smarter.

---

## ✨ Features

- 🖥️ **System Automation** — Open apps, manage files, run commands
- 🐍 **Python Scripting** — Execute and manage Python scripts
- 🔍 **Web Search & Research** — Fetch information from the web
- 📁 **File Management** — Create, find, and organize files
- 🌐 **GitHub Integration** — Full Git + SSH setup for seamless version control
- 🧠 **Smart Answers** — Knowledge retrieval and intelligent responses
- 🔬 **Advanced Research** — Scientific literature, databases, and more

---

## 🚀 Getting Started

### Prerequisites
- Windows 11
- Python 3.14+
- Git 2.55+

### Clone this Repository
```bash
git clone git@github.com:hariomsurve0-lab/assistant.git
cd assistant
```

### Run the Demo Script
```bash
python jarvis_demo.py
```
